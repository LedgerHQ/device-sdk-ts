## Who is this for?

The TypeScript SDK is aimed at JavaScript/TypeScript developers who want to integrate Ledger hardware device support into their applications:

- **Web applications** — via Web HID and Web BLE transports, for apps running directly in the browser.
- **Desktop applications** — via the Node HID transport, for Electron apps or Node.js scripts.
- **React Native applications** — via React Native BLE and HID transports, for cross-platform mobile apps in JavaScript.

It is the successor to the LedgerJS libraries (`hw-app-XXX`, `transport-XXX`) which are now deprecated. If you are migrating from LedgerJS, see the [LedgerJS: No longer maintained](./ledgerjs/info) section.
