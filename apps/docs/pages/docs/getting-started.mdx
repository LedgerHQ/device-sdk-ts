Welcome to the **Ledger Device Management Kit** documentation.

The DMK is a suite of libraries designed to simplify the integration of Ledger hardware devices into your applications — whether you're building for the web, desktop, or mobile.

Choose the SDK that matches your technology stack:

- [TypeScript SDK](/docs/typescript/introduction) — for web, desktop (Electron), and React Native applications.
- [Mobile SDK](/docs/mobile/introduction) — for native Android and iOS applications.

## Glossary

Throughout the documentation we use the following acronyms:

- **DMK**: Device Management Kit
- **DSK**: Device Signer Kit
